# 05computed计算属性

在Vue中如果使用一个数据信息，这个信息需要经过其他data做许多复杂处理后的才可以得出，那么可以通过computed计算属性实现，例如 加法计算器

使用computed计算属性的数据与访问data成员的数据语法都是一样的

语法：

var vm = new Vue({
  computed:{
    属性名称:function(){
      // 业务表达式实现,可以操作data成员数据
      return 返回结果
    } 
  }
})

计算属性特点：同时监听多个data数据的变化后，侧重于得到一个新的值；

注意：

  1. 每个计算属性都需要通过return关键字返回处理结果
  2. 计算属性关联的data如果发生变化，那么计算属性的方法也会重新计算并得出新值。

# 11Promise

# 概念

  • Promise表示一个异步操作，本身是一个构造函数，可以 new 出一个Promise的实例

  • 在Promise构造器接收函数作为参数，这个函数有两个参数 resolve(成功之后的回调函数)和 reject(失败后的回调函数)

    new Promise(function(resolve,reject){})

  • 在Promise构造函数的prototype属性上，有 .then()和catch() 两个方法

    then()接收成功执行结果 catch()接收失败执行结果

  • Promise创建的实例，会返回两种结果

    状态1：异步执行成功，需要在内部调用成功的回调函数resolve把结果返回给调用者 状态2：异步执行失败，需要在内部调用失败的回调函数reject把结果返回调用者

# 实例化对象

实例化Promise对象时，可以传递一个函数参数

var promise = new Promise(function () {
	// 异步执行过程
})

# then&catch

Promise构造器接受一个函数作为参数，这个函数有两个参数：resolve，reject， 分别代表这个Promise实例成功之后的回调函数和失败之后的回调函数。 举个例子：

var promise = new Promise(function (resolve, reject) {
    var a = 1
    if (a == 1) {
        resolve(a)
    } else {
        reject(error)
    }
})
promise.then(function (value) {
    console.log(value);
}).catch(function (error) {
    console.log(error);
})

// 输出：
// 1

# 语法糖

有时会看到项目中有如下效果应用

Promise.resolve()
Promise.reject()

两者都是是new promise（）的快捷方式。

Promise.resolve(value)是下面代码的语法糖:

new Promise(function (resolve) {
  resolve(value)
})

Promise.reject(value)是下面代码的语法糖:

new Promise(function(resolve,reject){
    reject(new Error("出错了"));
})

所以有如下应用

Promise.resolve(42).then(function(value){
    console.log(value);
});

// 等同于

var pro = new Promise(function (resolve) {
  resolve(42)
})
pro.then(function (value) {
  console.log(value)
})

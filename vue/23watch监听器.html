# 23watch监听器

watch监听：

vue的data数据部分可以被监听，数据一旦发生变化马上可以感知到，并做相关处理

watch 监听的特点：监听到某个数据的变化后，侧重于做某件事情；

只要被监听的数据发生了变化，会自动触发 watch 中指定的处理函数；

语法：

data(){
  return {
    name:'',
    addr:'',
    cat:{
      leg:'',
      tail:''
    }
  }  
}

watch:{
	data成员名称: 函数(变化后的值，变化前的值){}
  name:function(newval,oldval){},
  addr:function(newval,oldval){},
  'cat.leg':function(newval,oldval){},
  'cat.tail':function(){}
}

简单字符串成员可以监听，例如name、addr，复杂的对象，但是需要通过点连接被监听的成员(例如cat.leg和cat.tail)，并且要通过引号圈选

    这样通过this.$refs.xxx的方式可以找到

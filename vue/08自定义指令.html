# 08自定义指令

系统已经给我们提供了许多指令供使用，例如v-for v-bind v-model v-if v-show等等，如果这些还不满足，那么需要用户自己定义相关的指令，就是自定义指令

# 语法

官网参考

全局自定义指令定义语法：

Vue.directive('全局自定义指令名称', {  自定义指令配置对象  })

全局：当前Vue的全部实例都可以使用该指令

私有自定义指令定义语法：

<script>
    var vm = new Vue( {
        directives: {
            '私有自定义指令名称': { 配置对象 }
        }
    })
</script>

私有：只有指令所属的组件的Vue实例才可以使用该指令

# 配置对象

  • inserted 方法(使用指令的元素，被编译处理并渲染到容器后就调用)
inserted:function(el){
  // el:使用该指令的当前元素DOM对象，可以直接进行dom操作
}

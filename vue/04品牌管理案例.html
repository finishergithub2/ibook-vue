# 04品牌列表案例

# 列表展示&添加

添加品牌按钮

<input type="text" v-model="newbrand" />
<input type="button" value="添加" @click="add" />

品牌列表展示

<tr v-for="item in brand" :key="item.id">
  <td><input type="checkbox" /></td>
  <td>{{ item.id }}</td>
  <td>{{ item.name }}</td>
  <td>{{ item.ctime }}</td>
  <td><button>删除</button></td>
</tr>

vue实例中品牌列表数据和添加方法部分：

<script>
  var vm = new Vue({
    el: '#app',
    data: {
      newbrand: '', // 被增加的新品牌名称
      // 数组对象集
      brand: [
        { id: 10, name: '奔驰', ctime:Date.now() }, 
        { id: 11, name: '宝马', ctime:Date.now() }, 
        { id: 12, name: '奥迪', ctime:Date.now() }
      ],
      i: 12 // 已有品牌的序号基数
    },
    methods: {
      // 增加品牌
      add() {
        var obj = { id: ++this.i, name: this.newbrand, ctime:Date.now()  }
        this.brand.unshift(obj) // 给brand成员的前边添加一个新品牌
        this.newbrand = '' // 清空输入框的信息
      }
    }
  })
</script>

# 删除品牌

需要的核心方法：

数组.splice(下标, 长度)				 	//根据下标索引值删除数组指定长度的元素

定义删除按钮

<button href="#" @click="del(下标)">删除</button>

在事件声明中需要把被删除品牌的下标当成参数传递事件驱动中

删除逻辑

// methods方法中删除品牌实现逻辑
del( xiabiao ) {
  if (window.confirm('确认要删除该品牌么？')) {
    this.brand.splice(xiabiao, 1)
  }
},

步骤：

  1. 给每个品牌设置删除的按钮，同时传递被删除品牌的下标信息
  2. 在methods成员中设置方法，实现删除逻辑

# 筛选品牌

需要的核心方法：

// 从数组中把符合条件的单元筛选出来汇合成为新数组
数组.filter(item => item是否符合条件)
// 判断大字符串是否包含小字符串，并返回布尔值
大串.includes(小串)

双向数据绑定keywords成员

<input type="text" placeholder="请输入关键字" v-model="keywords" />

在vue的data成员中定义keywords成员

keywords: ''

在vue的computed成员中定义search属性(用以根据关键字筛选符合条件的信息)，内部通过keywords关键字实现数据筛选

methods:{
  guolv() {
    // 使得brand根据keywords做筛选
    // filter：从一个已有数组中通过条件筛选出一个"子级"数组
    //         本身有遍历机制
    //         通过返回值体现结果信息
    return this.brand.filter(v => {
      // 当前是“箭头函数”方法内部，本身的this与外部的this是同一个指引
      // v是小单元 如果符合条件，就return一个true的信息，那么这个小单元就汇总到新结果里边
      return v.name.includes(this.keywords)
    })
  }
}

数组.filter(callback回调函数) // 实现过滤(遍历)数组元素，同时把符合条件的元素以新数组形式返回

字符串.includes(小字符串) // 判断大字符串中是否包含小字符串，返回boolean值

在使用 v-for 做遍历时，不再面对data成员，相反是methods的guolv()方法

<tr v-for="item in guolv()" :key="item.id">
  <td><input type="checkbox" /></td>
  <td>{{ item.id }}</td>
  <td>{{ item.name }}</td>
  <td><button @click="del(item.id)">删除</button></td>
</tr>

v-for遍历guolv()方法返回值，必须设置()括号，表示函数调用，否则就被当做data数据成员了

步骤：

  1. 制作筛选输入框，双向数据绑定keywords
  2. 给Vue声明data成员，名称为keywords，用以感知输入的筛选关键字
  3. 通过methods的guolv()成员方法根据keywords实现过滤数据功能
  4. 页面上v-for直接遍历guolv()成员方法的返回值

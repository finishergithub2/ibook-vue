# 13品牌管理案例升级

通过所学内容(生命周期、axios) 对品牌管理案例做升级处理

# 品牌列表展示

通过生命周期created方法 和 methods的getBrandList实现首屏品牌数据的获取操作

步骤：

  1. 引入axios.min.js
  2. methods中声明getBrandList方法，以实现获取品牌数据
  3. 在created生命周期函数中直接调用getBrandList方法

  • 获得品牌列表数据

    // 获得品牌列表
getBrandList() {
  axios.get('http://127.0.0.1:3006/api/getprodlist').then(res => {
    if(res.data.status!==0){
      return alert('数据获取失败！')
    }
    this.brandList = res.data.message  // 把获得好的品牌结果赋予给brandList成员
  })
}

    axios请求完毕的返回结果是Promise对象，因此可以连贯调用then方法做进一步

  • created生命周期函数应用

    // 生命周期函数
created() {
  // 调用获得品牌列表的方法
  this.getBrandList()
}

    通过created生命周期函数调用getBrandList()方法，以便页面加载完毕就有对应的数据显示

# 添加品牌

思路：

在add()添加品牌的方法中，axios调用post方法向后端发请求，并把新品牌数据传递过去

添加完毕调用getBrandList()方法实现数据更新

// 增加品牌
add() {
  if(this.newbrand.trim().length!==0){
    axios.post('http://127.0.0.1:3006/api/addproduct',{name:this.newbrand}).then(res=>{
      if(res.data.status!==0){
        return alert('数据添加失败！')
      }
      this.getBrandList()  // 更新页面的数据
      this.newbrand = '' // 清空输入框的信息
    })
  }else{
    this.newbrand = '';   // 重置新品牌输入框
    return alert('请添加新品牌！')
  }
}

# 删除品牌

思路：

​ 在del()方法中，axios带着被删除品牌id，跑到服务器端实现删除操作

​ 删除完毕要调用getBrandList实现数据更新操作

// 删除品牌
del(id) {
  if (confirm('确认要删除该品牌么？')) {
    axios.get('http://127.0.0.1:3006/api/delproduct/'+id).then(res=>{
      if(res.data.status!==0){
        return alert('数据删除失败！')
      }
      this.getBrandList()
    })
  }
}

# 小结

  • 列表展示，getBrandList方法获得品牌列表信息、created生命周期函数调用getBrandList

  • 添加 ，添加表单域、v-model双向数据绑定、新品牌data成员newbrand、add添加方法、axios应用

  • 删除，获取被删除品牌id，删除按钮、del删除方法，axios应用

    添加、删除 品牌后可以重复调用 getBrandList()方法 以便更新页面数据
